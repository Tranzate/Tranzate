## 02 — Mécanique de grappin — Propulsion & Swing

<a href="https://www.youtube.com/watch?v=xiDiI-lUc-g">
  <img src="https://img.youtube.com/vi/xiDiI-lUc-g/maxresdefault.jpg" width="100%">
</a>

<table>
<tr>
<td width="60%">

Système de grappin **dual-mode** : une pression courte propulse le joueur, une pression longue déclenche un swing fluide. La détection du point d'accroche repose sur la position centrale de l'écran et la distance du joueur.

</td>
<td valign="top">

**Technologies**
- Unreal Engine
- Grappling Hook
- Physics
- Input System
- Raycast

</td>
</tr>
</table>

## 03 — Interface ImGui — Éditeur de niveau & Debug

<a href="https://www.youtube.com/watch?v=uZMzcTBj59o">
  <img src="https://img.youtube.com/vi/uZMzcTBj59o/maxresdefault.jpg" width="100%">
</a>

<table>
<tr>
<td width="60%">

Outil d'édition en runtime via **ImGui** avec un système de nodes pour piloter des éléments de niveau (plaques de pression, portes, ascenseur). Un second panneau expose les données du joueur en temps réel avec accès direct au Movement Component.

</td>
<td valign="top">

**Technologies**
- Unreal Engine
- ImGui
- Node System
- Debug Tools
- Movement Component

</td>
</tr>
</table>

## 04 — Jeu rythmique — Obstacles générés par la musique

<a href="https://www.youtube.com/watch?v=ybEjCUAJNks">
  <img src="https://img.youtube.com/vi/ybEjCUAJNks/maxresdefault.jpg" width="100%">

<table>
<tr>

</td>
